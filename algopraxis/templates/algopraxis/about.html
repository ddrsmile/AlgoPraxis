The idea of this website came from LeetCode.
I just thought that it must be cool that I could build my own online judgement website.
then ALGOPRAXIS was born.

The website was built with Django as the backend.
Celery and RabbitMQ have been implemented to run the codes with the worker.
Due to it's a personal project, RabbitMQ is installed in the same server.
But it is very easy to migrate RabbitMQ to another server if there is the needs in the future.

jQuery and Bootstrap were used to build the web pages.
As the latest update, not only Python3 but also JAVA and C++ have been implemented.
However, some utilities of C++ have not been implemented yet.
And therefore the future work of coderunner is to complete the implementation of utilities for C++.

Finally, Thank you very much for visiting ALGOPRAXIS.
